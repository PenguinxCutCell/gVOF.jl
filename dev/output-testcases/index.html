gVOF.printvoxel — Function
printvoxel(vg::VOFGrid, ifile::Int; dir=".", nvx=0, nvy=0, nvz=0)

Write a voxel-based volume-fraction field to voxel-<ifile>.vtk in VTK legacy-ASCII UNSTRUCTURED_GRID format.

For a uniform grid (igrid=1) the voxel grid matches the computational grid. For other grid types a uniform overlay is used.

gVOF.compdt! — Function
compdt!(vg::VOFGrid, params::VOFParams, timet::Float64) -> Float64

Compute the next time step dt from the CFL condition, with a 15 % growth limiter and a final-time guard. Returns the new dt and also stores it in params.dt conceptually (caller must use the returned value).
